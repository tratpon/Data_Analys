คู่มือการใช้ Excel Analysis ToolPak: Descriptive Statistics

หน้านี้อธิบายเป็นภาษาไทยขั้นตอนการใช้ฟีเจอร์ Descriptive Statistics ใน Excel Analysis ToolPak เพื่อสรุปสถิติพื้นฐาน (เช่น ค่าเฉลี่ย ส่วนเบี่ยงเบนมาตรฐาน ค่าต่ำสุด-สูงสุด ฯลฯ) โดยใช้ไฟล์ที่ได้จากแอป (`uploads/wine_input.xlsx`).

1) สรุปสั้น ๆ

  • Descriptive Statistics ให้สรุปตัวเลขสำคัญสำหรับแต่ละคอลัมน์ตัวแปร เช่น Mean, Median, Std. Dev, Min, Max, Count ฯลฯ
  • ใช้เมื่ออยากเข้าใจการกระจายของข้อมูลและตรวจสอบคุณภาพข้อมูลเบื้องต้น
  • ดู “ลักษณะทั่วไป” ของแต่ละตัวแปร เช่น ค่าเฉลี่ย ความเบี่ยงเบน การกระจาย

2) เตรียมไฟล์ Excel

  1. ดาวน์โหลด `wine_input.xlsx` จากหน้า index ของแอป โดยกรอกข้อมูลแล้วคลิก ดาวน์โหลด Excel
  2. เปิดไฟล์ใน Microsoft Excel และตรวจสอบว่าช่วงข้อมูลมีหัวคอลัมน์ และค่าทั้งหมดยังเป็นตัวเลข (หรือแปลงเป็นตัวเลขก่อน)

3) เปิดใช้งาน Analysis ToolPak (ถ้ายังไม่ได้เปิด)

ดูคำอธิบายในหน้า Correlation ถ้าคุณไม่แน่ใจวิธีเปิด

4) รัน Descriptive Statistics (ขั้นตอน)

  1. ไปที่ Data > Data Analysis
  2. เลือก Descriptive Statistics และกด OK
  3. กำหนดค่าดังนี้:
    • Input Range: ช่วงข้อมูลรวมหัวคอลัมน์ (ตัวอย่าง: A1:L51)
    • ติ๊ก Labels in first row ถ้าช่วงรวมหัวคอลัมน์
    • เลือก Grouped By: Columns (ข้อมูลแต่ละตัวแปรอยู่ในคอลัมน์)
    • เลือก Output Range (หรือ New Worksheet/Workbook)
    • ติ๊ก Summary statistics (ปกติจะติ๊กอันนี้เพื่อให้ได้ค่า Mean, StdDev, Median, Mode, Min, Max, Range, Count, ฯลฯ)
  4. กด OK — Excel จะสร้างตารางสรุปสำหรับแต่ละคอลัมน์

5) สิ่งที่ได้ (ตัวอย่างค่าพื้นฐาน)

Measureความหมาย
Meanค่าเฉลี่ย
Standard Errorประมาณค่าความคลาดเคลื่อนของค่าเฉลี่ย
Medianค่ากลาง
Modeค่าที่เกิดบ่อยที่สุด
Standard Deviationการกระจายของข้อมูล (ความแปรปรวน)
Sample Varianceความแปรปรวนตัวอย่าง
Kurtosis / Skewnessรูปร่างการแจกแจง (หาง/เอียง)
Range / Min / Maxค่าสูงสุด ต่ำสุด และช่วง
Count / Sumจำนวนตัวอย่าง และผลรวม

6) ตัวอย่างใช้งานจริง

  1. สมมติคุณมีคอลัมน์ `alcohol` และ `quality` ใน `wine_input.xlsx`
  2. เลือกช่วง A1:L51 (รวมหัวคอลัมน์) แล้วรัน Descriptive Statistics
  3. ดูค่า Mean, Std Dev ของ `alcohol` เพื่อประเมินสเกลและการกระจาย; หาก Std Dev สูง แปลว่าค่า alcohol กระจายกว้าง
  4. ดู Skewness เพื่อเช็คว่า distribution เอียงหรือไม่ — ถ้า skew มาก อาจต้องพิจารณาแปลงข้อมูลก่อนวิเคราะห์ต่อ

7) เคล็ดลับและข้อควรระวัง

  • ลบหรือจัดการค่าที่หายไปก่อนรัน (ToolPak จะคำนวณจากข้อมูลที่อยู่ในช่วงเท่านั้น)
  • หากต้องการสรุปเฉพาะบางคอลัมน์ ให้เลือกช่วงที่ประกอบด้วยคอลัมน์ที่ต้องการเท่านั้น
  • ค่าบางตัวเช่น Mode อาจไม่เหมาะสมกับข้อมูลต่อเนื่อง (continuous)
  • หากต้องการการวิเคราะห์เชิงลึก (p-value, CI) ให้ใช้สคริปต์ Python หรือ Add-in เสริม

8) ถัดไป

หากต้องการ ผม/เรา สามารถเพิ่ม:

  • ไฟล์ตัวอย่าง `uploads/wine_sample.xlsx` พร้อมข้อมูลตัวอย่างสั้น ๆ
  • สคริปต์ Python ใน `scripts/` เพื่อคำนวณ descriptive stats พร้อม p-values และ export เป็น Excel/CSV
  • visualization เบื้องต้น (histogram, boxplot) ด้วย Matplotlib/Seaborn

บอกผมว่าต้องการอย่างไหน แล้วผมจะเพิ่มให้เรียบร้อย
